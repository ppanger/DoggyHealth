# DoggyHealth 狗狗健康检测平台 - 设计规格文档

## 1. 项目概述

### 1.1 项目名称
DoggyHealth - 狗狗健康智能检测助手

### 1.2 项目类型
面向普通宠物主人的狗狗健康日常监测 Web 应用

### 1.3 核心价值主张
聚焦生活化场景：为担心狗狗健康隐患但不想去宠物医院或专业检测网站的普通用户提供高性价比、实用的日常健康检测工具。

### 1.4 目标用户
- 个人宠物主人（主要）：狗狗日常行为异常，想先初步排查
- 商业机构（次要）：宠物医院、宠物店可用于辅助初步筛查

---

## 2. 功能规划

### 2.1 核心功能（第一阶段）

#### 2.1.1 AI 图像健康检测
- **照片上传**：支持拖拽上传、点击上传
- **图像格式**：支持 JPG、PNG、WEBP，最大 5MB
- **AI 分析**：调用云端宠物健康垂直领域 API
- **分析维度**：
  - 整体外观状态（精神、毛发、体态）
  - 面部特征（眼睛、鼻子、嘴巴）
  - 行为姿态估算

#### 2.1.2 智能问答系统
- **动态问卷**：根据 AI 分析结果生成针对性的问题
- **问题类型**：
  - 饮食情况（食欲、饮水量）
  - 精神状态（活泼程度）
  - 排泄情况（呕吐、腹泻、粪便形态）
  - 行为变化（嗜睡、焦虑）
- **问题数量**：3-6 个问题（根据严重程度动态调整）

#### 2.1.3 健康报告生成
- **健康状态评估**：正常/需要关注/建议就医/建议立即就医
- **可能健康问题**：基于 AI 分析 + 问卷结果
- **专业建议**：分级的养护建议
- **温馨提醒**：免责声明、就医时机

### 2.2 扩展功能（第二阶段）

#### 2.2.1 健康记录
- **症状记录**：记录异常行为、症状描述
- **历史检测**：查看历史检测记录
- **趋势分析**：狗狗健康状态趋势图

#### 2.2.2 提醒中心
- **体检提醒**：设置定期体检提醒
- **疫苗提醒**：疫苗接种时间提醒
- **用药提醒**：日常用药提醒

#### 2.2.3 医疗资源
- **附近医院**：基于位置的宠物医院推荐（可选）
- **在线问诊**：第三方在线问诊入口（可选）

```
├── services/           # API 服务层
├── store/              # 状态管理
├── utils/              # 工具函数
├── styles/             # 全局样式
└── assets/             # 静态资源
```

### 3.3 模块设计

#### 3.3.1 图像分析模块
- **职责**：处理图片上传、预览、调用 AI API
- **接口**：
  - `uploadImage(file): Promise<AnalysisResult>`
  - `analyzeImage(imageData): Promise<AnalysisResult>`
- **依赖**：AI API 服务

#### 3.3.2 问卷引擎模块
- **职责**：根据分析结果动态生成问题
- **接口**：
  - `generateQuestions(analysisResult): Question[]`
  - `validateAnswers(answers): boolean`
- **依赖**：分析结果数据

#### 3.3.3 报告生成模块
- **职责**：综合分析结果与问卷答案，生成报告
- **接口**：
  - `generateReport(data): HealthReport`
  - `exportReport(report): void`
- **依赖**：问卷答案、分析结果

## 6. 数据模型

### 6.1 检测记录

```typescript
interface DetectionRecord {
  id: string;
  timestamp: number;
  imagePreview: string;
  analysisResult: AnalysisResult;
  answers: Record<string, string>;
  report: HealthReport;
}
```

### 6.2 分析结果

```typescript
interface AnalysisResult {
  state: '正常' | '可能不适' | '需要关注';
  confidence: number;
  symptoms: string[];
  possibleIssues: string[];
}
```

### 6.3 健康报告

```typescript
interface HealthReport {
  healthStatus: '正常' | '需要关注' | '建议就医' | '建议立即就医';
  severity: 0 | 1 | 2 | 3;
  possibleIssues: string[];
  recommendations: string[];
  createdAt: number;
}
```

## 7. 安全与隐私

- 图片仅用于当次分析，不持久化存储
- 本地仅保存脱敏后的检测记录（不含原图）
- 敏感数据加密存储
- 符合相关隐私法规
